# vue3-目录结构

- reactivity: 数据响应，独立，可与任何框架配合
- runtime-core --> 虚拟 DOM 渲染器、Vue 组件和 Vue 的各种API
- runtime-dom --> 处理原生 DOM API、DOM 事件和 DOM 属性
- runtime-test --> 序列化 DOM、触发 DOM 事件，以及记录某次更新中的 DOM 操作
- server-renderer 
- compiler-core 
- compiler-dom 
- shared: 平台无关的内部帮助方法
- vue
